This repository contains a simple interactive 3D environment built in C using the **raylib** library.

The program initializes a fullscreen window, loads a static 3D model (`City_1.glb`), and allows the user to navigate the scene in first-person mode.

---

## ‚ú® Features

* **First-Person Control:** Uses standard WASD keys for movement and mouse input for looking around.
* **3D Model Loading:** Demonstrates loading and drawing a GLB model (`resources/City_1.glb`).
* **Fullscreen Mode:** Initializes the application in fullscreen for a better immersive experience.
* **Optimized:** Uses a fixed target FPS of 80.
